## Packages
framer-motion | Essential for smooth, modern page transitions and interactive micro-animations
canvas-confetti | For delightful celebration effects on the result page
@types/canvas-confetti | TypeScript definitions for canvas-confetti

## Notes
- STRICTLY FRONTEND-ONLY: All data is loaded from `src/lib/mock-data.ts`.
- State is entirely managed via React state and `localStorage`.
- No backend API endpoints are called. The application acts as a fully self-contained static MVP.
- Using Wouter for routing.
- Custom fonts: Outfit (Headings) and DM Sans (Body) are configured via CSS variables.
